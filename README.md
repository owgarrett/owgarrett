### What Up 👋

This repo now includes a runnable MATLAB project for your NI-DAQ sensor workflow at:

- `sensor_test_system/`

Start in MATLAB with:

```matlab
cd('sensor_test_system')
addpath(genpath(pwd))
sensor_test_gui
```

For a no-hardware dry run:

```matlab
run('examples/demo_end_to_end.m')
```
